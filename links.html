webdev

  • Solaria's Guide to Accessibility - an incredibly simple & in-depth guide to understanding accessibility
  • Semantic HTML - tutorial/guide for converting a webpage to use semantic html
  • Tessisamess - themes & templates made primarily for insanejournal, but can be used anywhere html allows
  • Hint.css - css only tooltip library that works with <span> for all your tooltip needs
  • CSS Validation - W3C's css validator service
  • Markup Validation - W3C's markup (markdown and html) validation service
  • Webcode.tools - a website with cheatsheets for html, css, and js elements
  • HTML Cheatsheet - a bunch of essential generators for webdev elements
  • HTML Dog - basic tutorials for simple html, beginner friendly
  • Neocities RSS Guide - simple guide to make a custom rss feed for neocities
  • Tiny Tools Directory - a massive directory of various small and open source tools
  • Clip Path Maker - tool for making cool shapes to use around css images and boxes
  • Unicode Directory - reference site for unicode symbols, emojis, and icons
  • Google Webfont Helper - a resource to replace using google's font directory
  • ImageOptim - online image optimizer to decrease file sizes
  • CSS Gradient - generate css gradients easily without having to undergo trial and error
  • Font Squirrel Webfont Maker - a tool to turn your favourite fonts into webfonts for self hosting
  • Transfonter - alternate tool to create and self-host custom webfonts
  • Fontdrop - tool for viewing what's inside fonts and comparing different typefaces
  • How To Center In CSS - what it says on the tin
  • Quackit - yet another html and css reference guide with assorted tutorials
  • Speech Bubble Generator - tool to easily create css-only speech bubbles
  • Responsinator - site for checking out how your site looks on difference devices
  • Text Stroke Generator - for getting that perfect stroke around text since css is hard
  • Dithermaker - one of many dithering tools, useful for creating decker-style images
  • Dither Me This - alternate dithering tool that seems easier to use, with a more eye-pleasing design
  • Your Fonts - free handwritten font maker alternative to calligraphr
  • Meta Tags - site to generate meta tags for your site
  • Tiny Helpers - similar to tiny tools but with lots more resources for developers
  • Image Map - quick-n-easy image map maker
  • Shark Coder - yet another reference site with useful css tricks and tips
  • Border Image Generator - MDN web docs border generator tool
  • Accessible Color - tool to anaylse and create more accessible patterns (used for this site)
  • Free Code Camp - amazing free resource to learn all kinds of web development and gain certificates from courses
  • Bechnokid's Freezeframe.js - accessible friendly freeze frame script to pause moving images that i just couldn't figure out how to work just yet
  • Transparent Textures - tool that converts subtle patterns into and generates transparent textures for use
  • Tiled Backgrounds - bunch of tiled/repeating backgrounds from sadgirl.online--the entire site is a useful resource
  • Large Gallery Tutorial - for making LARGE galleries with js or jquery
  • Deploy To Neocities Tutorial - the clearest possible tutorial on how to use the deploy to neocities github action + implement version control for your site
